Basics Linux Commands:

echo hi ------> Hi (Print statement as in Py)
ls ---> used to list files and folders in a directory
cd my_dir1  -----> changes the directory

pwd ---> presents the working directory

mkdir new_directory ----> this helps in making us a new directory

so for example if we do the below operations:

cd new_directory;mkdir www;pwd 
it open the new_directory and then it will make a new directory named as www and then show us the the directory by www
