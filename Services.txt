Services in Linux:

1) service httpd start : start HTTPD service
2) systemctl start httpd

> systemctl stop httpd -stops the server

> systemctl status httpd check httpd service status

> systemctl enable httpd configure httpd to start at startup

> systemctl disable httpd configure htppd to not start at startup
